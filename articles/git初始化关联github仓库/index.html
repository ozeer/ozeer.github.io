Git初始化关联github仓库

By ozeer

October 30, 2022
1、引入某个项目文件夹，执行git init
2、执行git remote -v，查看当前文件夹是否已和远端仓库有关联
3、添加远程仓库关联
 git remote add origin git@github.com:ozeer/laravel-api.git
4、再次执行git remote -v，发现已经有关联了
origin  git@github.com:ozeer/laravel-api.git (fetch)
origin  git@github.com:ozeer/laravel-api.git (push)
